# Miteinander Android DEV

Aktuelle manuell installierbare Testversion der Miteinander-Android-App.

Version: 0.1.35-dev

[Miteinander-dev.apk herunterladen](https://raw.githubusercontent.com/DrMilos33/Miteinander-Android-DEV/main/Miteinander-dev.apk)

Die APK installiert keine Updates selbst und fordert keine Berechtigung zum Installieren anderer Apps an. Android kann bei einer manuellen Installation trotzdem auf die externe Quelle hinweisen.

Zum Aktualisieren die vorhandene DEV-App nicht deinstallieren, sondern die neue APK öffnen und **Aktualisieren** wählen. Lokale App-Daten bleiben dadurch erhalten.

Prüfsumme und Buildinformationen stehen in `SHA256SUMS.txt` und `build-info.json`. Der Quellcode liegt im privaten Entwicklungsrepository.
